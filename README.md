## 前言

本史书旨在系统性地将作者得到的抽象的某潮 495 条按时间顺序梳理为更具体但也更抽象的、某潮官方以及社区的光荣历史，同时尽可能附上来源或相关讨论，为大家在对线时提供更加可靠的火力支撑，并让大家更深刻清晰地了解某神游以及其神人玩家的光辉历史。

本史书攻击性可以保证，因为来史书底下找本人对线的潮 U，从其发言可以看出，没有一个有勇气仔细看到最后（或者说直面真相）。

笔者使用的史料来源或许不够清晰准确和客观，但为学习并发扬极端潮 U 不顾逻辑只要自己觉得有攻击性就开岁的优良作风，笔者也就不挑论据了。

对线时有人问笔者，玩个游戏有必要像打仗一样吗？有，因为对于飞到眼前的蚊子和蟑螂有必要往死里拍。
